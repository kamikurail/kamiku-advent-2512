# kamiku-advent-2512 (Angular sample)

Minimal Angular + TypeScript sample app.

Prerequisites:
- Node.js 16+ (or compatible with the installed TypeScript/Angular versions)
- npm

Install and run:

```bash
npm install
npm start
```

Notes:
- This is a minimal scaffold for local development. No HTTP/backend setup is included.
- package.json (dependencies/devDependencies) was added at the project root.
